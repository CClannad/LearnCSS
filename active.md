# :Active

1. :link

   未访问过的样式

   ~~~css
   a:link {
       color: blue;
   }
   ~~~

   

2. :active

   点击的样式

   ```css
   a:active {
       color: red;
   }
   ```

3. :hover

   悬停的样式

   ```css
   a:hover {
       color: #000;
       text-decoration: underline overline;
   }
   ```

4. :visisted

   访问过的样式

```
a:visited {
    color: chartreuse;
}
```
